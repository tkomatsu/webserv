仏発エンジニア養成機関「42 Tokyo」が広島県の呉工業高等専門学校で講演活動を実施

一般社団法人42 Tokyo（本社：東京都港区、代表理事 亀山敬司、以下 42 Tokyo（ https://42tokyo.jp/）は、2021年7月11日（日）に呉工業高等専門学校で講演活動を実施いたしました。



42 Tokyoは、2020年6月に開校した、学費無料、経歴不問の「誰でも平等に挑戦できる、教育機会の提供」をミッションに掲げた、フランス発のエンジニア養成機関です。

学生同士が学びあう「ピア・ラーニング」方式を採用し、コミュニケーション活性化のため、当初、オフラインでの学習を前提として準備を進めていました。しかし、世界的な新型コロナウイルス感染拡大を受け、オンラインで学習できる環境整備を行いました。
Piscineや入学後の学習をすべてオンライン化することで、居住地などによる距離の問題を飛び越え、首都圏以外の学生も参加可能になりました。(合格者の居住地の属性はこちら）

そこで今回、首都圏外の学生にも新たな学習の選択肢として42 Tokyoがあることを知ってもらうため、第1弾として、広島県の呉工業高等専門学校で講演活動を実施し、講演当日は呉工業高等専門学校の1、2年生320名を対象に42 Tokyoに関する説明を行いました。


42 Tokyoは、今年開校から２年目を迎えます。今後も環境に左右されない教育と、挑戦の機会提供の場として、今回のような講演活動も継続的に行ってまいります。
