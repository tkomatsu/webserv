個人情報保護に関して

個人情報保護方針

一般社団法人42Tokyo（以下「当校」といいます。）は、事業を実施する上で、個人情報の重要性を認識し、個人情報を適切に取扱うことが社会的責務であると考えます。
そこで、当校は、個人情報の適切な取扱いについて組織として取り組むために個人情報保護方針を以下のとおり定め、従業員に周知し、個人情報の保護に努めます。

1．法令の遵守について

当校は、個人情報の適切な取扱いのため、個人情報の保護に関する法律及び関連法令を遵守します。

2．安全管理措置について

当校は、個人情報の紛失、改ざん、漏えい等を防止するため、適切な安全管理措置を講じます。
また、個人情報を取り扱う従業員及び委託先に対して、適切な監督を行います。

3．個人情報の利用について

当校は、個人情報をあらかじめ通知・公表した利用目的の範囲内でのみ利用します。

4．個人情報の第三者への提供について

当校は、法令に基づく場合を除き、ご本人の事前の同意なく、個人情報を第三者に提供しません。

5．個人情報の開示、訂正等について

当校は、ご本人から個人情報の開示、訂正等を求められた場合は、ご本人であることを確認し、適切に対応します。

個人情報の取り扱いについて

一般社団法人42Tokyoの生徒及びお取引様等（以下「利用者様」といいます。）の個人情報の取り扱いに関して、「個人情報の保護に関する法律」に基づき、以下の事項を公表します。

1．個人情報取扱事業者の名称

一般社団法人42Tokyo

2．個人情報の利用目的

  • （1）各種研修・イベント・セミナー・サービス提供のため
  • （2）スポンサーシップ、サービス代金等の決済・請求のため
  • （3）利用者様の本人確認のため
  • （4）利用者様からのお問い合わせに対応するため
  • （5）アフターサービスを提供するため
  • （6）当校及び他社の各種研修・イベント・セミナー・サービスの情報提供（電子メールの送付及びカタログ・チラシの郵送）のため
  • （7）メンテナンスその他の重要なお知らせのため
  • （8）アンケート調査・分析及びマーケティング調査・分析のため
  • （9）広告の表示のため
  • （10）各種研修・イベント・セミナー・サービス改善のため
  • （11）当校で開催される各種研修・イベント・セミナー・サービスの不正利用の防止のため
  • （12）利用者様の与信判断、債権回収のため
  • （13）利用者様との取引に関する事務を行うため
  • （14）利用者様に対し必要な連絡を行うため
  • （15）上記利用目的に付随する利用のため
  • （16）その他当校の利用規約等で定める目的のため

上記以外の利用目的で個人情報を取得し、利用する場合は、あらためて当校ホームページ等で公表します。

3．個人情報の第三者への提供

当校は、個人情報の保護に関する法律その他法令で利用者様の同意が不要とされている場合を除いては、利用者様の同意を得た場合（第三者提供する旨が記載された規約等に同意してサービスの申込を行った場合を含みます。）に限り、個人情報を第三者（外国にある第三者を含みます。）に提供します。

4．個人情報の委託先への提供

当校は、利用目的の達成に必要な範囲内で、個人情報の取り扱いの全部又は一部を第三者（外国にある第三者を含みます。）に委託する場合があります。

5．個人情報の共同利用

当校は、利用目的の達成に必要な範囲内で、個人情報を共同利用します。

  • (1)共同して利用される個人情報の項目氏名、ニックネーム、ハンドルネーム、メールアドレス、電話番号、保証人に関する情報、住所、生年月日、性別、アカウント情報、サービス利用履歴、その他サービス提供中に当校が利用者様に関して取得する全ての個人情報
  • (2)共同して利用する者の範囲
    合同会社DMM.com
  • (3)共同して利用する者の利用目的
    「2．個人情報の利用目的」に掲げた利用目的と同一
  • (4)共同して利用する個人情報の管理について責任を有する者の名称
    一般社団法人42Tokyo

6．個人情報の開示等のお問い合わせ先

当校は、個人情報の開示等のご要望があったときは、所定の手続きで利用者様本人であることを確認のうえ、対応します。

  • ・個人情報の開示、訂正、利用停止、消去等のお問い合わせ先
  • ・個人情報の取り扱いに関する苦情のお申し出先
連絡先：42Tokyo個人情報お問合せ窓口（メール）
メールアドレス：privacy@42tokyo.jp
受付時間：24時間、年中無休

7．改定

当校は、【個人情報の取り扱いについて】を、適宜見直し、予告なく、改定することがあります。改定は、当校ホームページ等で公表した時点から適用されるものとします。
